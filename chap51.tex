\section{代表値}

代表値には以下のものがあります。
\begin{itemize}
  \item 中心
    \begin{itemize}
      \item 中央値
      \item (算術)平均値
      \item モード(最頻値)
    \end{itemize}
  \item モーメント
    \begin{itemize}
      \item 2次のモーメント：分散 $\to$ 標準偏差
      \item 3次のモーメント：歪度(分布の非対称性)
      \item 4次のモーメント：尖度(裾の厚さ、頂点の鋭さ)
    \end{itemize}
  \item 多変量解析
    \begin{itemize}
      \item 共分散・相関係数
      \item 多変量共分散
      \item 回帰分析
      \item 因子分析
      \item 主成分分析
      \item クラスター分析
      \item 多次元尺度法
    \end{itemize}
\end{itemize}
