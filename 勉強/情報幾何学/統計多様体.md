# 統計多様体
$M$を有限次元実$C^\infty$多様体、$g$を$M$上のリーマン計量、$T$を$M$上の3階対称共変テンソルとする。これらの組$$(M,g,T)$$を **統計多様体** という。

# 統計多様体の射
$(M_1,g_1,T_1)$、$(M_2,g_2,T_2)$を統計多様体とする。$C^\infty$写像$f:M_1\to M_2$が以下を満たす時、 **統計多様体の射** という。
- 等長写像である：$f^*g_2=g_1$
- 統計構造を保つ：$f^*T_2=T_1$

# 同値な定義
リーマン多様体$(M,g)$上に3階対称共変テンソル$T$を設定することと、[[α接続]]を設定することは同値である。実際、$\alpha$接続$\nabla^{(\alpha)}$が存在するとき、$$
T_{ijk}:=\frac2{\alpha}(\Gamma_{ijk}^{(0)}-\Gamma_{ijk}^{(\alpha)})
$$で3階対称共変テンソルが定まる。α接続から定まる統計多様体を$$(M,g,\nabla^{(\alpha)})$$と書く。
