# 測度
$(\Omega,\mathcal{F})$を[[可測空間]]とする。写像$\mu:\mathcal{F}\to\mathbb{R}$が$\Omega$上の **測度** であるとは、次を満たす時を言う。
- $\mu(\varnothing)=0$
- $\{A_n\}\subset\mathcal{F}$が高々可算個の族のとき、$A_n\cap A_m=\varnothing$ ($\forall n,m$ s.t. $n\neq m$)ならば$$\mu\left(\bigcup_nA_n\right)=\sum_n\mu(A_n)$$をみたす。

# 測度空間
可測空間$(\Omega,\mathcal{F})$と、その測度$\mu$の組$(\Omega,\mathcal{F},\mu)$のこと。
