# 定義
$(X,\mathcal{O}_X)$を[[環付き空間]]、$\mathcal{F}, \mathcal{G}$を$\mathcal{O}_X$[[加群の層]]とする。[[前層]]$$\mathbf{Top}(X)\ni U\mapsto \mathcal{F}(U)\otimes_{\mathcal{O}_X(U)}\mathcal{G}(U)\in\mathbf{Ab}$$の[[層化]]を、$\mathcal{F}$と$\mathcal{G}$のテンソル積といい、$\mathcal{F}\otimes_{\mathcal{O}_X}\mathcal{G}$で書き表す。
