# 定義
$f:X\to Y$を[[スキームの射]]とする。このとき、$f$が局所有限であるとは次を満たす時を言う。ある$Y$の[[スキーム|アフィン開被覆]]$V_i=\operatorname{Spec}B_i$が存在して、任意の$i$に対して$f^{-1}(V_i)$は$X$のアフィン開部分集合$U_{ij}=\operatorname{Spec}A_{ij}$で被覆される。ここで任意の$i,j$に対して、$A_{ij}$は有限生成$B_i$代数である。

## Note
ほぼ有限次元多様体のこと。ただし特異点をかなり許容する。
