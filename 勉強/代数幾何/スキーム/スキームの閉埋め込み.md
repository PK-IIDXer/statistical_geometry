# 定義
[[スキームの射|射]]$f:X\to Y$が閉埋め込みであるとは、次を満たす時を言う。
- $f$によって$\mathrm{sp}(X)$は$\mathrm{sp}(Y)$の閉部分集合に同相である
	- $\mathrm{sp}(X)$は[[スキーム]]$X=(X,\mathcal{O}_X)$の底空間$X$のこと
- $Y$上の[[局所環付き空間の射|層に誘導される射]]$f^\#:\mathcal{O}_Y\to f_*\mathcal{O}_X$は[[層の全射|全射]]である
