# 定義
環$A$の素イデアルの集合$\operatorname{Spec}A$に[[Zariski位相]]を入れ、以下の[[層]]$\mathcal{O}$を備えた[[局所環付き空間]]を、$A$のスペクトラムという
- $s\in\mathcal{O}(U)$は$U$から$\bigsqcup_{\mathfrak{p}\in U}A_\mathfrak{p}$への写像であり、$s(\mathfrak{p})\in A_\mathfrak{p}$ ($\forall\mathfrak{p}\in U$)をみたす。
- 任意の$\mathfrak{p}\in U$に対して、その開近傍$V$であって$U$に含まれるものと、$a,f\in A$が存在して、任意の$\mathfrak{q}\in V$に対して、$f\notin\mathfrak{q}$かつ$s(\mathfrak{q})=a/f\in A_\mathfrak{q}$をみたす。
	- Note: これは多様体上の正則関数と全く同じ書き方
